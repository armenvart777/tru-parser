# tru-parser

Парсер каталога ТРУ (Технические средства реабилитации) с выгрузкой результатов в Google Sheets.

## Что делает

Скачивает перечень товаров из каталога на сайте госзакупок (ktsr.sfr.gov.ru), сопоставляет позиции из вашей таблицы с каталогом по модели и производителю, парсит страницы товаров — достаёт код ОКПД2, номер регистрационного удостоверения и ссылку на скан РУ. Если товар не найден через ТСР, подключается к API Росздравнадзора и ищет там.

Результат — CSV-файл и выгрузка в указанный Google Sheets.

## Запуск

```bash
pip install -r requirements.txt
```

Положите `credentials.json` (сервисный аккаунт Google) в папку проекта.

```bash
# Полный цикл: собрать каталог → сматчить → спарсить → выгрузить
python main.py --run

# Только собрать каталог (сохраняется в кэш)
python main.py --collect

# Только матчинг (каталог должен быть в кэше)
python main.py --match

# Тест-парсинг одного товара по коду
python main.py --test 12-34-56.123
```

## Конфиг

В `config.py` задаются:
- `SPREADSHEET_ID` — ID таблицы Google Sheets
- Задержки между запросами (`REQUEST_DELAY`)
- Пути к файлам прогресса и результатов

## Стек

Python 3.11+, aiohttp, google-api-python-client
